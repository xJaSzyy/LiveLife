# 🌱 Live Life

## 🧩 О игре

Live Life — это головоломка с живой физической симуляцией, где игрок не строит мир с нуля, а аккуратно вмешивается в уже существующую экосистему.

Задача — дорисовывать уровни, изменяя ландшафт так, чтобы достичь цели, не разрушив хрупкий баланс мира.

Каждое действие имеет последствия: изменение рельефа влияет на поток воды, поведение существ и общую стабильность системы.
